# 广东海洋大学校园网自动登录脚本（2025.11修复版）

一个用于广东海洋大学校园网自动登录的 Tampermonkey（油猴）脚本，可自动填写账号密码并完成登录操作，节省手动登录的时间。

## 功能说明

- 自动识别广东海洋大学校园网登录页面（`http://10.129.1.1/*`）
- 自动填写预设的账号和密码
- 自动点击同意协议并提交登录请求
- 修复原版本因域名变更导致的失效问题

## 安装步骤

1. 确保你的浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 插件（支持 Chrome、Firefox、Edge 等主流浏览器）
2. 点击 [安装脚本](https://还没上传.js) 或手动将脚本添加到 Tampermonkey
3. 安装完成后，编辑脚本，将 `user_account` 和 `user_password` 的值替换为你的校园网账号和密码

## 使用方法

1. 连接广东海洋大学校园网（未登录状态下会自动跳转到登录页面）
2. 脚本会自动检测登录页面并执行登录操作，无需手动干预
3. 登录过程会在浏览器控制台输出相关信息（可按 F12 打开控制台查看）

## 配置修改

修改脚本中的以下变量来设置你的账号信息：
```javascript
// 替换为你的校园网账号
var user_account = '校园网账号';
// 替换为你的校园网密码
var user_password = '校园网密码';
```

## 注意事项

- 本脚本仅适用于广东海洋大学校园网特定登录页面，其他学校或页面可能不适用
- 请妥善保管你的账号密码，建议仅在个人设备上使用
- 若登录页面结构发生变化（如按钮选择器变更），脚本可能需要相应调整
